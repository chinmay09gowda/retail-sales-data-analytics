# Retail Sales Data Analytics Dashboard

This project analyzes retail sales data using Python and Power BI to discover insights such as sales trends, regional performance, and top-selling products. The project demonstrates data cleaning, exploratory data analysis, and dashboard creation for business decision-making.


## Project Objective

The objective of this project is to analyze retail sales data and visualize key business insights using data analytics techniques. The dashboard helps businesses understand sales performance, identify profitable products, and analyze regional sales trends.



## Technologies Used

Python  
Pandas  
NumPy  
Matplotlib  
Seaborn  
Power BI  


## Dataset

Superstore Sales Dataset

The dataset contains retail transaction data including:

Order ID  
Order Date  
Product Name  
Category  
Region  
Customer Segment  
Sales  
Profit  

Dataset Source:
https://www.kaggle.com/datasets/vivek468/superstore-dataset-final

## Project Features

Data Cleaning and Preprocessing  
Exploratory Data Analysis (EDA)  
Sales Trend Analysis  
Region-wise Sales Analysis  
Product Category Performance  
Power BI Interactive Dashboard  


## Power BI Dashboard

The Power BI dashboard provides insights such as:

Total Sales and Profit  
Sales by Region  
Sales by Product Category  
Monthly Sales Trends  
Top Performing Products
